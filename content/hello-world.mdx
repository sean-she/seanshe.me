## HTML and CSS Examples

You can now use raw HTML in your MDX files! Here are some examples:

### Inline Styles

<div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); padding: 2rem; border-radius: 8px; color: white; margin: 1rem 0;">
	<h3 style="margin-top: 0;">Styled with inline CSS</h3>
	<p>This div uses inline CSS for styling!</p>
</div>

### HTML Elements

<button style="background-color: #4CAF50; color: white; padding: 10px 20px; border: none; border-radius: 4px; cursor: pointer; font-size: 16px;">
	Click me!
</button>

### Custom Styled Content

<div style="border-left: 4px solid #667eea; padding-left: 1rem; margin: 1.5rem 0;">
	<p style="margin: 0; font-style: italic; color: #666;">

		CSS.
	</p>
</div>
